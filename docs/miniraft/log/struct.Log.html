A collection of LogEntries

Fields§

§entries: Vec<LogEntry<T>>

Log entries

§committed_len: LogIndex

How much of the log has been considered committed. A log entry is considered ‘safely replicated’ or committed once it is replicated on a majority of servers. Only meaningful on servers which are leaders. Initialized to 0 as no entries are committed. Increases monotonically.

§applied_len: LogIndex

How much of the log has been applied to the state machine. Initialized to 0, increases monotonically.

§app: Box<dyn App<T, S>>

State machine

§parent_id: ServerId

ServerId of our parent for pretty printing documentation
